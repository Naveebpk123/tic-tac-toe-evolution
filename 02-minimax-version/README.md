# Phase 2: CLI With Minimax AI

This is the second phase of **Tic-Tac-Toe Evolution**.

Phase 2 keeps the terminal-based gameplay from phase 1, but adds a minimax-powered AI opponent. It also keeps local 2-player mode available.

## Features

- Play against a minimax AI
- Continue playing in 2-player local mode
- Terminal-based game flow
- Same board and input style as phase 1
- Win, loss, and draw detection

## Files

- `main.py`: CLI game flow for player-vs-player and player-vs-AI
- `minimax.py`: minimax algorithm and helper functions

## How To Run

To run this release:

```bash
cd 02-minimax-version
python main.py
```

## Notes

This release includes both local 2-player mode and minimax AI mode.
